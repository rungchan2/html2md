# Privacy Policy

**html2md Chrome Extension**
**Last Updated**: December 12, 2025
**Version**: 1.0.0

---

## 1. Overview

html2md is a Chrome extension that allows users to visually select specific sections of web pages and convert them to Markdown format. This Privacy Policy explains how html2md handles user information.

---

## 2. Information Collection and Use

### 2.1 Information We DO NOT Collect

html2md does **NOT collect** the following information:

- ❌ Personally identifiable information (name, email, address, etc.)
- ❌ Browsing history
- ❌ Website visit history
- ❌ Search queries
- ❌ Location data
- ❌ Cookies or tracking data
- ❌ User web activity data

### 2.2 Locally Stored Data

html2md stores the following data **locally on your device only**:

- ✅ **Conversion History**: Markdown content you have converted (maximum 10 items)
  - Storage Location: Chrome local storage (`chrome.storage.local`)
  - Storage Purpose: To allow quick access to previously converted content
  - Retention Period: Until you delete it or uninstall the extension
  - External Transmission: **NOT transmitted**

---

## 3. Data Security

- All data processing occurs locally within your browser
- No data is transmitted to external servers
- No remote code execution
- No third-party analytics tools are used

---

## 4. Permission Usage

html2md requests the following permissions, each used only for its stated purpose:

### 4.1 Host Permissions
- **Not Used**: Does not request broad host permissions
- **Uses activeTab Instead**: Only operates on tabs where you explicitly activate the extension

### 4.2 activeTab
- **Purpose**: Access user-selected content on the currently active tab
- **When Used**: When you click the extension icon or use the keyboard shortcut

### 4.3 clipboardWrite
- **Purpose**: Copy converted Markdown to clipboard
- **When Used**: Only when you click the "Copy" button

### 4.4 downloads
- **Purpose**: Download converted Markdown as .md files
- **When Used**: Only when you click the "Download" button

### 4.5 scripting
- **Purpose**: Dynamically inject selection UI into web pages
- **When Used**: Only when you activate selection mode

### 4.6 sidePanel
- **Purpose**: Display conversion history in side panel
- **Data Scope**: Only locally stored conversion history

### 4.7 storage
- **Purpose**: Store conversion history locally (maximum 10 items)
- **Storage Location**: Your device only

---

## 5. Third-Party Sharing

html2md does **NOT share any data with third parties**:

- ❌ Advertising networks
- ❌ Analytics services
- ❌ External servers
- ❌ Developers or operators

---

## 6. User Rights

You have the following rights:

### 6.1 Data Access
- View stored conversion history in the side panel

### 6.2 Data Deletion
- Delete individual items: Click the delete button for each item in the side panel
- Delete all history: Click the "Clear All" button in the side panel
- Complete deletion: Uninstall the extension

### 6.3 Data Export
- Copy feature: Copy each conversion history to clipboard
- Download feature: Download each conversion history as .md file

---

## 7. Children's Privacy

Since html2md does not collect personal information, it is safe for users of all ages, including children under 13.
