# 개인정보처리방침 (Privacy Policy)

**html2md Chrome Extension**
**최종 업데이트**: 2025년 12월 12일
**버전**: 1.0.0

---

## 1. 개요

html2md는 사용자가 웹 페이지의 특정 섹션을 시각적으로 선택하여 마크다운 형식으로 변환할 수 있는 Chrome 확장 프로그램입니다. 본 개인정보처리방침은 html2md가 사용자 정보를 어떻게 처리하는지 설명합니다.

---

## 2. 정보 수집 및 사용

### 2.1 수집하지 않는 정보

html2md는 다음과 같은 정보를 **수집하지 않습니다**:

- ❌ 개인 식별 정보 (이름, 이메일, 주소 등)
- ❌ 브라우징 기록
- ❌ 웹사이트 방문 기록
- ❌ 검색 쿼리
- ❌ 위치 정보
- ❌ 쿠키 또는 추적 데이터
- ❌ 사용자의 웹 활동 데이터

### 2.2 로컬 저장 데이터

html2md는 다음 데이터를 **사용자의 기기에만 로컬로 저장**합니다:

- ✅ **변환 기록**: 사용자가 변환한 마크다운 콘텐츠 (최대 10개)
  - 저장 위치: Chrome 로컬 스토리지 (`chrome.storage.local`)
  - 저장 목적: 사용자가 이전에 변환한 콘텐츠를 빠르게 다시 접근할 수 있도록 함
  - 보관 기간: 사용자가 삭제할 때까지 또는 확장 프로그램 제거 시까지
  - 외부 전송: **전송되지 않음**

---

## 3. 데이터 보안

- 모든 데이터 처리는 사용자의 브라우저 내에서 로컬로 이루어집니다
- 어떠한 데이터도 외부 서버로 전송되지 않습니다
- 원격 코드 실행이 없습니다
- 제3자 분석 도구를 사용하지 않습니다

---

## 4. 권한 사용

html2md는 다음 권한을 요청하며, 각 권한은 명시된 목적으로만 사용됩니다:

### 4.1 호스트 권한
- **사용하지 않음**: 광범위한 호스트 권한을 요청하지 않습니다
- **대신 activeTab 사용**: 사용자가 명시적으로 활성화한 탭에서만 작동

### 4.2 activeTab
- **목적**: 현재 활성 탭의 사용자 선택 콘텐츠 접근
- **사용 시점**: 확장 프로그램 아이콘 클릭 또는 단축키 사용 시

### 4.3 clipboardWrite
- **목적**: 변환된 마크다운을 클립보드에 복사
- **사용 시점**: 사용자가 "복사" 버튼을 클릭했을 때만

### 4.4 downloads
- **목적**: 변환된 마크다운을 .md 파일로 다운로드
- **사용 시점**: 사용자가 "다운로드" 버튼을 클릭했을 때만

### 4.5 scripting
- **목적**: 웹 페이지에 선택 UI 동적 삽입
- **사용 시점**: 사용자가 선택 모드를 활성화했을 때만

### 4.6 sidePanel
- **목적**: 변환 기록을 사이드 패널에 표시
- **데이터 범위**: 로컬 저장된 변환 기록만

### 4.7 storage
- **목적**: 변환 기록을 로컬에 저장 (최대 10개)
- **저장 위치**: 사용자의 기기 내부만

---

## 5. 제3자 공유

html2md는 **어떠한 데이터도 제3자와 공유하지 않습니다**:

- ❌ 광고 네트워크
- ❌ 분석 서비스
- ❌ 외부 서버
- ❌ 개발자 또는 운영자

---

## 6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

### 6.1 데이터 확인
- 사이드 패널에서 저장된 변환 기록 확인 가능

### 6.2 데이터 삭제
- 개별 기록 삭제: 사이드 패널에서 각 항목의 삭제 버튼 클릭
- 전체 기록 삭제: 사이드 패널에서 "전체 삭제" 버튼 클릭
- 완전 삭제: 확장 프로그램 제거

### 6.3 데이터 내보내기
- 복사 기능: 각 변환 기록을 클립보드로 복사
- 다운로드 기능: 각 변환 기록을 .md 파일로 다운로드
