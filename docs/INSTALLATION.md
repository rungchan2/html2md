# html2md 설치 및 사용 가이드

## 설치 방법

### 1. Chrome Extension 로드하기

1. Chrome 브라우저에서 `chrome://extensions/` 페이지를 엽니다
2. 우측 상단의 **"개발자 모드"**를 활성화합니다
3. **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 클릭합니다
4. 이 프로젝트의 루트 디렉토리(`/Users/heechan/Desktop/Heechan/project/html2md`)를 선택합니다
5. html2md 확장 프로그램이 설치됩니다

### 2. 설치 확인

- Chrome 툴바에 파란색 "M" 아이콘이 표시됩니다
- 아이콘을 클릭하면 팝업이 나타납니다

## 사용 방법

### 기본 사용 흐름

1. **선택 모드 활성화**
   - Extension 아이콘 클릭, 또는
   - `Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows) 단축키

2. **섹션 선택**
   - 마우스를 움직여 원하는 섹션에 호버하면 파란색 하이라이트 표시
   - 클릭하여 섹션 선택 (진한 파란색 테두리로 확정 표시)

3. **내보내기**
   - 선택 후 나타나는 팝업에서:
     - 📋 **복사**: 클립보드에 마크다운 복사
     - 💾 **다운로드**: `.md` 파일로 다운로드

### 고급 기능

#### 다중 선택
- `Shift+클릭`으로 여러 섹션을 선택할 수 있습니다
- 선택된 섹션들은 `---`로 구분되어 변환됩니다

#### 범위 조절
- `Shift+↑`: 부모 요소로 선택 범위 확대
- `Shift+↓`: 자식 요소로 선택 범위 축소

#### 취소
- `ESC` 키를 누르면 선택 모드가 종료됩니다
- 아이콘을 다시 클릭해도 토글됩니다

## 지원 HTML 요소

변환되는 HTML 요소들:

- **헤딩**: `<h1>` ~ `<h6>` → `#` ~ `######`
- **리스트**: `<ul>`, `<ol>` → `-`, `1.`
- **코드블록**: `<pre><code>` → ` ```언어 `
- **인라인 코드**: `<code>` → `` ` ``
- **테이블**: `<table>` → 마크다운 테이블
- **링크**: `<a>` → `[text](url)`
- **이미지**: `<img>` → `![alt](url)`
- **강조**: `<strong>`, `<b>` → `**text**`
- **이탤릭**: `<em>`, `<i>` → `*text*`
- **인용구**: `<blockquote>` → `>`

## 파일명 규칙

다운로드되는 파일명 형식:
```
{페이지_제목}_{YYYYMMDD}.md
```

예시: `React_Documentation_20251211.md`

## 문제 해결

### Extension이 로드되지 않는 경우
- manifest.json 파일이 프로젝트 루트에 있는지 확인
- 모든 필수 파일이 있는지 확인
- Chrome 개발자 모드가 활성화되어 있는지 확인

### 선택 모드가 작동하지 않는 경우
- 페이지를 새로고침하고 다시 시도
- Chrome Console(F12)에서 에러 메시지 확인
- lib 폴더에 turndown.js 파일들이 있는지 확인

### 마크다운 변환이 이상한 경우
- 일부 사이트는 특수한 HTML 구조를 사용합니다
- 선택 범위를 `Shift+↑/↓`로 조절해보세요
- 더 작은 단위로 선택해서 복사하세요
