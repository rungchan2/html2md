Chrome 웹스토어 개인 정보 보호 관행 폼 답변 (업데이트)
=================================================

중요: host_permissions 제거, activeTab만 사용으로 변경됨
검토 지연 문제 해결됨

1. 단일 목적 설명 (Single Purpose Description)
---------------------------------------------------
html2md는 웹 페이지의 특정 섹션을 시각적으로 선택하여 마크다운 형식으로 변환하는 도구입니다. 사용자가 문서, 블로그, 기술 문서 등의 콘텐츠를 마크다운으로 쉽게 저장하고 복사할 수 있도록 돕습니다.


2. 원격 코드 사용 이유 (Remote Code Justification)
---------------------------------------------------
이 확장 프로그램은 원격 코드를 사용하지 않습니다. 모든 코드는 확장 프로그램 패키지에 포함되어 있으며, 외부에서 코드를 불러오거나 실행하지 않습니다.


3. 호스트 권한 사용 이유 (Host Permissions)
---------------------------------------------------
광범위한 호스트 권한을 요청하지 않습니다. activeTab 권한만을 사용하여 사용자가 명시적으로 확장 프로그램을 활성화한 탭에서만 작동합니다. 이는 Chrome의 권장 사항을 따르며 사용자 보안을 향상시킵니다.


4. activeTab 권한 사용 이유
---------------------------------------------------
사용자가 확장 프로그램을 명시적으로 활성화한 탭에서 HTML 콘텐츠에 접근하여 마크다운으로 변환하기 위해 필요합니다. 확장 프로그램 아이콘 클릭 또는 단축키 사용 시에만 작동하여 사용자 제어를 보장합니다.


5. clipboardWrite 권한 사용 이유
---------------------------------------------------
변환된 마크다운을 사용자의 클립보드에 복사하는 기능을 제공하기 위해 필요합니다. 사용자가 "복사" 버튼을 클릭했을 때만 클립보드에 쓰기 작업이 수행됩니다.


6. downloads 권한 사용 이유
---------------------------------------------------
변환된 마크다운을 .md 파일로 다운로드하는 기능을 제공하기 위해 필요합니다. 사용자가 "다운로드" 버튼을 클릭했을 때만 파일 다운로드가 실행됩니다.


7. scripting 권한 사용 이유
---------------------------------------------------
사용자가 확장 프로그램을 활성화할 때 웹 페이지에 선택 UI와 변환 기능을 동적으로 삽입하기 위해 필요합니다. activeTab과 함께 사용되어 사용자가 명시적으로 허용한 탭에서만 작동합니다.


8. sidePanel 권한 사용 이유
---------------------------------------------------
변환 기록을 사이드 패널에 표시하여 사용자가 이전에 변환한 마크다운 콘텐츠를 쉽게 확인하고 관리할 수 있도록 하기 위해 필요합니다. 모든 기록은 로컬에만 저장됩니다.


9. storage 권한 사용 이유
---------------------------------------------------
최근 변환한 마크다운 최대 10개를 사용자의 브라우저 로컬 스토리지에 임시 저장하여 변환 기록 기능을 제공하기 위해 필요합니다. 저장된 데이터는 사용자의 기기를 벗어나지 않으며, 언제든지 삭제할 수 있습니다.


============================================================
주요 변경사항:
============================================================
✅ host_permissions (<all_urls>) 제거됨
✅ content_scripts 제거됨 (동적 삽입으로 대체)
✅ activeTab 권한만 사용 - Chrome 권장 사항 준수
✅ 사용자가 명시적으로 활성화한 탭에서만 작동
✅ 검토 지연 문제 해결됨

이제 확장 프로그램이 더 안전하고 빠른 검토를 받을 수 있습니다.
